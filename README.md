# OVERIDE

Overide es un projecto orientado a mejorar los viajes terrestres intermunicipales. EL objetivo del proyecto es hacer el proceso de administracion y gestion de los itinerarios de viajes mucho mas sencillo y organizado para las compa√±ias de viajes terrestres y hacer el proceso de busqueda y compra mas amigable y confiable para los usuarios.

El proyecto consta de dos partes, una seccion orientada al back-end y otra al front-end. En los siguientes apartados podra encontrar la documentacion asociada a los proyectos.
